Checksheet Manager Documentation¶

Welcome to Checksheet Manager ‘s documentation. Checksheet Manager is a free online checksheet management system for educational instiutions.

The code is open source, and available on github. Any issues regarding installation or use of checksheet manager can be reported at the issues page.
